# GO WEED WEB - Angular Task Manager

Administrador de tareas visual inspirado en tableros tipo Asana, desarrollado en Angular.

## Funciones incluidas

- 5 proyectos: Nativo, Universidad de Manizales, Freelance, Estudio Empírico y Personal.
- Vista calendario, tablero, lista y bandeja.
- Creación de tareas desde botón general o desde un día del calendario.
- Panel lateral de detalle al hacer clic en una tarea.
- Edición de título, descripción, fecha, hora, prioridad, estado y proyecto.
- Comentarios por tarea.
- Botón de finalizar con celebración de hojas, flores y cohetes.
- Botón Compartir funcional con copiar enlace e invitar colaboradores.
- Botón Personalizar funcional con temas, vista compacta y celebraciones.
- Menú de usuario funcional para Jacobo Ospina Gallo.
- Persistencia en localStorage.

## Cómo ejecutarlo

```bash
npm install
npm start
```

Luego abre:

```bash
http://localhost:4200/
```

Si ya estás dentro de la carpeta del proyecto, no vuelvas a ejecutar `cd nombre-carpeta`. Ejecuta directamente `npm install` y luego `npm start`.
